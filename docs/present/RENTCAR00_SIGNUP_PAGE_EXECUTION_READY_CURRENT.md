## 0. 이번 단계 목적

현재 단계는 **회원가입 페이지 폼 자체를 만드는 것**이다.

구현 위치:
- `src/pages/SignupPage.jsx`

구현 방식:
- 지금 `/signup` 에 있는 이메일/비밀번호/비밀번호확인 폼을 유지한다.
- 그 페이지를 확장해서 아래 항목을 **밑으로 이어서 붙인다.**

추가할 항목:
- 이름
- 생년월일
- 휴대폰 번호
- OTP 영역
- 주소 영역
- 약관 동의 영역
