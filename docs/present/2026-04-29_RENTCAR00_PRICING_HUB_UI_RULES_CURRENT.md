# RENTCAR00 PRICING_HUB UI rules current

Last updated: 2026-04-29

이 문서는 관리자 UI를 어떻게 줄일지 잠근다.

---

## 1. 화면 목표

관리자는 아래만 하면 된다.

1. 그룹 선택
2. 기존 IMS 기준값 확인
3. 24h 기준금액 수정
4. 기간 percent 정책 추가
5. 실제 적용 금액 preview 확인

---

## 2. 숨길 것

아래는 1차 active 화면에서 제거 또는 숨김 대상이다.

- legacy JSON 카드
- 범용 rate_scope 직접 편집
- 복잡한 override 입력
- 내부 metadata 노출
- 채널 추상화 용어

---

## 3. 보여줄 것

### 상단
- 그룹명
- IMS 그룹 ID
- 연결 정책명
- 기준값 불러오기 상태

### 기본값 편집
- 24시간 기준금액

### 기간 정책
- 기간명
- 시작일 / 종료일
- 적용 요일
- percent 조정값
- 활성화 여부

### 계산 preview
- 기본 24h
- 기간 적용 후 24h
- 계산된 6h / 12h / 추가시간
- 계산된 주간 / 장기 금액
- IMS 반영 예상값
- 찜카 반영 예상값

---

## 4. 편집 원칙

1. 24h 절대금액만 직접 입력 가능
2. 기간은 percent 만 직접 입력 가능
3. 나머지 금액은 읽기 전용 결과값으로 표시
4. 저장 전 preview 와 저장 후 preview 가 같은 계산 규칙을 써야 함

---

## 5. 한 줄 결론

PRICING_HUB UI는 **입력은 좁게, 결과는 실제 금액 중심으로 넓게** 보여주는 방향으로 잠근다.
