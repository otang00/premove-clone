# 08. 완료 기준

## 완료로 보는 조건
- 메인 예약 페이지 외형이 완성되어 있다
- 차량 목록 페이지 외형이 완성되어 있다
- 차량 상세/예약 페이지 외형이 완성되어 있다
- 더미데이터로 화면이 자연스럽게 채워진다
- 주요 버튼/탭/선택 UI가 시각적으로 동작한다
- 페이지 이동 흐름이 최소 수준으로 연결된다
- 데스크톱 기준 시연 가능 상태다

## 아직 미완료로 남겨도 되는 것
- 실재고 계산
- 예약 생성
- 문자 인증
- 결제 처리
- 운영자 페이지

## 최종 산출물 성격
- ‘작동하는 프로토타입 UI’ 직전의 설계 문서
- 2단계 기능 연결을 위한 안전한 뼈대

## 한 줄 판정 기준
사용자가 봤을 때 ‘메인에서 시작해 예약 흐름이 이미 있는 것처럼 보이면’ 1단계 합격.
