# 03. 디자인 시스템

## 레이아웃 원칙
- 데스크톱 우선 설계
- 최대 콘텐츠 폭 고정형 래퍼 사용
- 헤더 sticky
- 상세 페이지는 2컬럼 구조

## 핵심 시각 요소
- 흰 배경 카드
- 연한 회색 배경 섹션
- 둥근 모서리 카드
- 연보라/보라 계열 포인트 컬러
- 얇은 회색 보더
- 정보성 텍스트는 중간 회색

## 토큰 초안
- background: #ffffff / #f5f5f7 계열
- border: #e5e7eb 계열
- primary: 보라 계열 1종
- text-main: 진회색
- text-sub: 중간 회색
- danger/alert는 1단계에서 최소화

## 타이포 규칙
- 페이지 타이틀: 강조형
- 섹션 타이틀: 중간 강조
- 본문: 14~16px 중심
- 보조 문구: 12~13px
- 가격: 굵고 크게

## 인터랙션 원칙
- hover/focus 상태는 단순하게
- disabled 버튼 스타일 정의
- 실제 동작보다 클릭 가능한 외형 표현 우선

## 지도 처리
- 1단계에서는 실제 지도 SDK 의존 최소화
- 지도 영역은 이미지 또는 박스 placeholder로 대체 가능
